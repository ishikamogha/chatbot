## Overview

```javascript
const chatbot = {
  type    : "Interactive Browser Chatbot",
  stack   : ["HTML5", "CSS3", "Vanilla JavaScript"],
  ui      : "Responsive chat interface with smooth message transitions",
  logic   : "JavaScript-based query handling and response system",
  status  : "Backend integration in progress",
  goal    : "Simulate AI-driven conversational responses"
};
```

A clean, browser-based chatbot featuring real-time message flow, smooth CSS animations, and a custom JavaScript query-handling engine. The frontend is complete and fully responsive. Backend integration is actively in development to enable intelligent, AI-powered responses.

## Getting Started

```bash
# Clone the repository
git clone https://github.com/ishikamogha/chatbot.git

# Navigate into the project
cd chatbot

# Open directly in browser
open index.html
```

> No `npm install`. No build steps. Open and interact.
