
## Getting Started

```bash
#  Clone the repository
git clone https://github.com/ishikamogha/chatbot.git

#  Navigate into the project
cd chatbot

#  Open in your browser
open index.html
```

> No `npm install`. No build steps. Open and chat!
